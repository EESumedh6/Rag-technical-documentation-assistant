FastAPI Beginner to Intermediate Guide

Introduction

FastAPI is a modern Python framework for building APIs with Python.
It is fast, lightweight, and easy to use.
FastAPI is based on standard Python type hints.

Main Features

- High performance
- Automatic validation
- Interactive API docs
- Async support
- Easy integration with Pydantic
- Clean syntax
- Built-in OpenAPI support

Installation

Install FastAPI and Uvicorn.

pip install fastapi uvicorn

Creating First FastAPI App

from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def home():
    return {"message": "Hello FastAPI"}

Running Application

uvicorn main:app --reload

Open Browser

http://127.0.0.1:8000

Swagger Documentation

http://127.0.0.1:8000/docs

ReDoc Documentation

http://127.0.0.1:8000/redoc

GET Routes

GET routes are used to retrieve data.

Example

@app.get("/users")
def get_users():
    return ["Alice", "Bob"]

GET Route with Dictionary

@app.get("/status")
def status():
    return {"status": "running"}

POST Routes

POST routes create new resources.

@app.post("/users")
def create_user():
    return {"message": "User created"}

PUT Routes

PUT routes update existing resources.

@app.put("/users/{user_id}")
def update_user(user_id: int):
    return {"updated_user": user_id}

DELETE Routes

DELETE routes remove resources.

@app.delete("/users/{user_id}")
def delete_user(user_id: int):
    return {"deleted_user": user_id}

Path Parameters

Path parameters are dynamic values in URLs.

@app.get("/users/{user_id}")
def get_user(user_id: int):
    return {"user_id": user_id}

Example URL

/users/5

Multiple Path Parameters

@app.get("/users/{user_id}/posts/{post_id}")
def get_post(user_id: int, post_id: int):
    return {
        "user_id": user_id,
        "post_id": post_id
    }

Query Parameters

Query parameters are optional values.

@app.get("/search")
def search(q: str):
    return {"query": q}

Example

/search?q=fastapi

Default Query Parameters

@app.get("/items")
def items(skip: int = 0, limit: int = 10):
    return {
        "skip": skip,
        "limit": limit
    }

Optional Query Parameters

from typing import Optional

@app.get("/products")
def products(category: Optional[str] = None):
    return {"category": category}

Request Body

FastAPI uses Pydantic models for request validation.

from pydantic import BaseModel

class User(BaseModel):
    name: str
    age: int

@app.post("/users")
def create_user(user: User):
    return user

JSON Request Example

{
  "name": "John",
  "age": 25
}

Response Models

Response models define API response structure.

@app.get("/profile", response_model=User)
def profile():
    return {
        "name": "Alice",
        "age": 30
    }

Nested Models

class Address(BaseModel):
    city: str
    country: str

class Customer(BaseModel):
    name: str
    address: Address

List Fields

from typing import List

class Blog(BaseModel):
    tags: List[str]

Validation

FastAPI automatically validates request data.

Invalid data returns validation errors.

Status Codes

FastAPI supports HTTP status codes.

from fastapi import status

@app.post("/create", status_code=status.HTTP_201_CREATED)
def create():
    return {"message": "Created"}

Common Status Codes

200 OK
201 Created
400 Bad Request
401 Unauthorized
403 Forbidden
404 Not Found
500 Internal Server Error

Error Handling

Use HTTPException for errors.

from fastapi import HTTPException

@app.get("/items/{item_id}")
def get_item(item_id: int):
    if item_id != 1:
        raise HTTPException(
            status_code=404,
            detail="Item not found"
        )

    return {"item_id": item_id}

Middleware

Middleware runs before and after requests.

from fastapi import Request

@app.middleware("http")
async def log_requests(request: Request, call_next):
    print("Request received")

    response = await call_next(request)

    print("Response sent")

    return response

Middleware Use Cases

- Logging
- Authentication
- Monitoring
- Performance tracking

Dependency Injection

Dependencies help reuse logic.

from fastapi import Depends

def common_parameters():
    return {"version": "1.0"}

@app.get("/info")
def info(params: dict = Depends(common_parameters)):
    return params

Authentication Dependency

def verify_token(token: str):
    if token != "secret":
        raise HTTPException(
            status_code=401,
            detail="Unauthorized"
        )

@app.get("/secure")
def secure(token: str = Depends(verify_token)):
    return {"message": "Authorized"}

Async Routes

FastAPI supports asynchronous routes.

@app.get("/async")
async def async_route():
    return {"message": "Async route"}

Use Async For

- Database calls
- External APIs
- Long-running tasks

Tags

Tags organize API endpoints.

@app.get("/users", tags=["Users"])
def users():
    return []

APIRouter

Large applications use routers.

from fastapi import APIRouter

router = APIRouter()

@router.get("/products")
def products():
    return []

app.include_router(router)

CORS Middleware

from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

File Uploads

from fastapi import UploadFile, File

@app.post("/upload")
def upload(file: UploadFile = File(...)):
    return {"filename": file.filename}

Form Data

from fastapi import Form

@app.post("/login")
def login(username: str = Form(...)):
    return {"username": username}
